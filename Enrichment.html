  • 2019 Summer:Summer Software Internship at Molekiu Limited

    • 2019 Summer: Summer Software Internship at Molekiu Limited

    I built a recpetion app for a dispalying robot, in Java aand with Android Studio and the robot SDK. The aim was to develop an app so that it can move the robot, and do whatever a receptionalist can do. This includes, leading the visitors to different places of the office, signing in and out for visitors and informing vistors about upcoming events. Apart from this, we've also implemented tts and voice command abilities to the app, making it more interactive. Apart from that, I also built a backend website using php, sql, html and css for the museum app. The goal of the museum app is to allowing a robot to do whatever or even better than what a museum guide can do. It should be able to explain and provide media contents when it arrived at certain marker. The website that I worked on allows users to upload explanations and customized medias contents for different markers. Lastly, I was also part of the team that designs the course materials for their stem program. The plan is to teach children code in java, so that they can eventually create programs for our robots. It was a very fruitful and fun experience overall.

I also built an Image Classifier in Pycharm with Tensorflow in Ubuntu, and trained it with the MNIST dataset, which is a dataset that contains a bunch of handwritten numbers.
